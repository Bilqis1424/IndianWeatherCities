# Old Versions of Model Scripts

This folder contains earlier versions of the forecasting scripts developed during the research process.

These files are retained for transparency, version tracking, and reproducibility purposes. They reflect intermediate stages of model development, experimentation, and refinement prior to the final implementation used in the manuscript.

## Purpose
- Track the evolution of the modelling framework  
- Provide transparency in the development process  
- Support reproducibility and verification of results  

## Note
The final and validated implementation used in this study is available in the main repository directory (see main.py).

Users are advised to refer to the main script for reproducing the results presented in the manuscript.
